# ProjectLens Interview OS

Contributor: `rishabh-gt-hub`

This is a separate frontend contribution added inside the `InterviewX` repository to reflect Rishabh's work.

## What it includes

- React + Vite frontend
- Login page, home page, dashboard, records, about, and contact
- Git repository URL input for repo-aware mock interview prompts
- Browser voice transcription support
- Camera preview support
- Mock performance analytics and improvement insights
- Local previous-session tracking

## Run locally

```bash
npm install
npm run dev
```

Run these commands inside:

`contributions/rishabh-gt-hub/projectlens-interview-os`
